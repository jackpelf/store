Easy_encrypt

Sep 16th, 2015 5:36 pm

0x01 移位密码

e(x) = (x + K) mod 26 d(y) = (y - K) mod 26 若K=3，则称为凯撒密码

0x02 置换密码

e(x) = pi(x) d(y) = pi-1(y)

0x03 仿射密码

e(x) = ( ax + b ) 只有gcd(a, 26) = 1才是单射函数。 与m互素的个数用φ(m)表示，称为欧拉函数 密钥空间为mφ(m)

0x04 维吉尼亚密码

e(x1, x2, …, xn) = (x1 + k1, x2 + k2, …, xn + kn) d(y1,y2, …, yn) = (y1 - k1, y2 - k2 , …, yn - kn)

0x05 希尔密码

(y1, y2,…, yn) = (x1, x2,…, xn)(k11, k12, …k1n) (k21, k22, … ) ( ) (kn1, kn2, …, knn) e(y) = xK d(x) = yK-1 如果密钥K可逆，则加解密可完成

0x06 置换密码

置换打乱了明文字母的位置和次序。 e(x1, …, xn) = (xπ(1), …, xπ(m)) d(y1, …, yn) = (y-1π(1), …, y-1π(n)) 置换密码是希尔密码的一种特殊情形，K是每行每列只有一个1，其他都是0的矩阵。

0x07 流密码

连续的明文使用相同的密钥K来加密，称为分组密码 产生一个密钥流来加密明文，称为流密码 同步流密码 最简单的流密码直接由初始化密钥使用某种算法变换得来，密钥流和明文串是相互独立的，这种流密码称为同步流密码 对于所有i>=1的整数有zi+d = zi，则称该流密码为具有周期d的流密码 流密码通常以二元字符表示，加密解密都可看作摸2的加法 e(x) = (x+z)mod2 d(y) = (y+z)mod2 密钥流zi的产生不但与密钥K有关，而且还与明文元素或密文元素有关，这类流密钥称为异步流密码 自动密钥密码使用明文来构造密钥流（除了最初的原始密钥） e(x) = (x+z)mod26 d(y) = (y-z)mod26
